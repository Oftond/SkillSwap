# SkillSwap — Платформа для обмена навыками

**SkillSwap** — это внутренняя площадка, где пользователи могут предлагать свои навыки (например, «научу играть на гитаре») и получать за это виртуальную валюту (коины), чтобы оплачивать услуги других участников.

## Основной функционал (MVP)
- **Авторизация**: Безопасный вход и регистрация пользователей.
- **Лента объявлений**: Просмотр всех услуг с фильтрацией по категориям (Музыка, Программирование, Языки и др.).
- **Личный кабинет**: Управление профилем, просмотр баланса коинов и своих объявлений.
- **Система заявок**: Покупка услуг с автоматическим списанием и начислением виртуальной валюты.

## Технологический стек

### Backend
- **Framework**: Laravel 11 (PHP 8.2+)
- **Database**: PostgreSQL
- **Authentication**: JWT (JSON Web Tokens)
- **API Style**: RESTful API

### Frontend
- **Library**: React 18+
- **Build Tool**: Vite
- **HTTP Client**: Axios
- **Routing**: React Router DOM
- **Styling**: Tailwind CSS
- **Notifications**: React Hot Toast
